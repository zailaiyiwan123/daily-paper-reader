## 摘要
最先进的文本转语音（TTS）模型实现了令人印象深刻的自然度和表现力，然而，对说话风格的细粒度、解耦控制仍然具有挑战性。在电影配音、游戏配音和视频内容生成等专业场景中，用户通常需要在保留所有其他风格的同时，修改特定的风格类别，如情感、年龄或性别。现有的风格可控TTS方法通常依赖于文本描述的风格或语音参考风格迁移，这使得难以同时控制显式语义属性并保留微妙的、文本未描述的韵律细节。我们提出了AutoSIFT，一个用于类别级风格编辑的可控语音生成框架。AutoSIFT将说话风格分解为已知的文本可描述类别和捕捉非言语韵律及说话者特定细微差别的未知残余风格。它由一个广义风格解耦器（从参考语音中提取类别感知的风格原型）和一个任意风格填充器（从参考语音中选择性地填充未指定的风格类别）组成。通过仅替换文本指定的风格类别同时保留残余的语音派生风格，AutoSIFT实现了自然、富有表现力且高度可定制的语音生成。

## Abstract
State-of-the-art text-to-speech (TTS) models achieve impressive naturalness and expressiveness, yet fine-grained, disentangled control over speaking styles remains challenging. In professional scenarios such as film dubbing, game voice acting, and video content generation, users often need to modify a specific style category, such as emotion, age, or gender, while preserving all others. Existing style-controllable TTS methods typically rely on either text-described styles or speech-reference style transfer, making it difficult to jointly control explicit semantic attributes and preserve subtle, text-undescribed prosodic details. We propose AutoSIFT, a controllable speech generation framework for category-level style editing. AutoSIFT decomposes speaking style into known text-describable categories and unknown residual styles that capture non-verbal prosody and speaker-specific nuances. It consists of a generalized Style Disentangler, which extracts category-aware style prototypes from reference speech, and an Arbitrary Style Infiller, which selectively infills unspecified style categories from the reference. By replacing only text-specified style categories while preserving residual speech-derived styles, AutoSIFT enables natural, expressive, and highly customizable speech generation.

## 1. 核心问题与整体含义（研究动机和背景）

- **核心问题**：现有最先进的文本转语音（TTS）模型虽然自然度和表现力出色，但难以实现对说话风格的**细粒度、解耦控制**。在电影配音、游戏配音、视频内容生成等专业场景中，用户常需修改特定风格类别（如情感、年龄、性别），同时保留其他所有风格（如音色、节奏、呼吸等非语言韵律）。
- **现有方法的不足**：
  - **文本提示方法**（如 PromptTTS）通过自然语言描述控制风格，但无法捕捉精细声学变化（如音色、韵律微变化）。
  - **语音提示方法**（如语音克隆）虽能保留丰富声学信息，但将风格编码为整体表示，导致类别级编辑困难。
  - **多模态/解耦方法**（如 CLAP、ParaMETA）试图结合文本可控性与语音表现力，但存在分类丢弃残余风格信息、类别干扰和负迁移等问题。
- **研究意义**：提出一种既能修改用户指定的风格类别，又能保留参考语音中未指定和残余风格的可控语音生成新范式。

## 2. 方法论：核心思想与关键技术细节

### 核心思想
- **任意风格填充（Arbitrary Style Infilling, ASI）任务**：给定文本提示（指定任意子集风格类别）和参考语音（包含丰富声学风格信息），生成语音时遵循文本指定的类别，同时继承参考语音中所有未指定和残余风格。
- **风格分解**：将说话风格分解为**已知文本可描述类别**（如性别、年龄、情感、语言）和**残余风格因素**（如细微音色、微韵律、说话质感等难以用离散标签描述的特征）。

### 关键技术细节（三阶段分层训练）

#### 阶段1：基础风格可控TTS骨干训练
- **Style Extractor（风格提取器）**：基于Transformer编码器+注意力池化，从语音嵌入中提取全局风格嵌入S（192维），输入到Flow-Matching扩散Transformer（FM-DiT）中，实现内容与风格解耦。
- **训练目标**：流匹配损失，迫使风格提取器捕获所有必要风格信息用于高保真语音重建。

#### 阶段2：风格解耦器（Style Disentangler）
- **风格分解**：使用N+1个类别特定线性投影器（ψ_c1,...,ψ_cn, ψ_r）将全局风格嵌入S映射到d维子空间，得到每个类别的嵌入s_ci和残余嵌入s_r。
- **原型对齐损失（PAL）**：为每个类别维护一组可学习原型（语义质心），通过余弦相似度检索最近原型，并拉近s_ci与其对应原型距离，实现语义对齐。未标注样本使用检索伪标签。
- **残差量化**：对残余嵌入s_r进行向量量化（VQ），正则化残差空间。
- **风格重建**：将量化后的类别原型与残差嵌入拼接，通过风格融合器重建原始风格嵌入，使用MSE损失确保信息完整。
- **总损失**：L_disen = L_recon + L_PAL + L_VQ。

#### 阶段3：任意风格填充器（Arbitrary Style Infiller, ASI）
- **门控筛选**：对于每个文本可描述类别c_i，若文本提示指定了该类别，则使用文本派生嵌入；否则使用语音派生的原型。残余风格始终继承自语音。
- **风格融合**：将所有选定的类别嵌入与残差嵌入拼接，通过线性风格融合器生成最终风格嵌入S_hat_f。
- **训练策略**：**审查式学习（Censored Learning）**——训练中以25%概率随机丢弃类别文本标签，使模型学会动态切换到语音分支。
- **损失**：风格重建损失L_asi = ||S_hat_f - sg(S)||^2。

### 公式/算法流程（文字说明）
- 输入：参考语音Y → 风格提取器 → 全局风格S。
- 文本描述t → 文本编码器 → 文本风格嵌入T。
- 风格解耦：S → {s_ci, s_r}（通过投影器）；T → {t_ci}。
- ASI门控：f_ci = t_ci（若指定）否则sg(s_ci)；残差始终sg(s_r)。
- 融合：concat所有f_ci和sg(s_r) → 线性风格融合器 → 重建风格S_hat_f。
- 最终生成：将S_hat_f作为条件输入TTS骨干，生成语音。

## 6. 主要结论与发现

- **AutoSIFT成功实现了任意风格填充任务**：能够仅修改文本指定的风格类别，同时高保真保留参考语音中未指定类别和残余声学细节（如音色、韵律微变化）。

## 8. 不足与局限

- **风格类别覆盖有限**：当前仅处理预定义的四种类别（性别、年龄、情感、语言），无法处理更细粒度或连续属性（如口音、语速、音高范围、对话意图等）。
- **依赖类别标注**：原型学习需要类别级标注，标注覆盖度和一致性可能影响解耦质量。未来可探索弱监督或自动发现风格维度的方法。
- **残余风格定义模糊**：残余风格是“文本不可描述”的集合，其模型表示（量化码本）可能未完全捕获所有细微差异，缺乏独立评估。
- **生成质量仍有提升空间**：在中文文本提示下PER达14.32，高于英文（3.59），可能受限于数据不平衡或多语言对齐。
- **扩展性挑战**：添加新类别需重新训练风格解耦器和填充器，增量学习或零样本扩展未讨论。
- **仅进行了客观和主观MOS评估**，未进行大规模用户研究或实际应用测试（如配音效果的主观评价）。
