## 摘要
随着内容平台的爆炸式增长，推荐系统需要更好地满足用户需求，以提升用户满意度和留存率。以短视频平台为例，用户不仅追求高质量内容，还期望找到值得信赖的作者。尽管生成式推荐系统近年来取得了突破，但现有方法主要生成单一类型的推荐内容，并且通常采用低效的自回归范式来生成语义ID。本文提出了一种名为HGenPush的端到端异构生成式推荐架构。首先，我们设计了一个混合用户行为理解模块，该模块融合了多场景、多视角的行为，以捕捉精准的用户兴趣。然后，我们设计了一个双分支异构生成式推荐模块，将视频推荐和作者推荐集成到统一框架中。此外，为了提升生成效率，我们设计了一种轻量级的多令牌预测方法，摒弃了自回归范式。最后，我们设计了一个用户消费偏好对齐模块，利用用户反馈作为奖励信号，引导模型生成更高质量的内容，从而提升用户体验和参与度。通过这些设计，HGenPush同时满足了用户对高质量内容和可信赖作者的需求。我们已在大型短视频平台快手的推送通知系统中部署了HGenPush，使日活跃用户数显著提升了0.181%。

## Abstract
With the explosive growth of content platforms, recommendation systems need to better satisfy user demands to enhance user satisfaction and retention. Taking short-video platforms as an example, users not only seek high-quality content but also trusted authors. Although generative recommendation systems have achieved breakthroughs in recent years, existing methods primarily generate single-type recommendation content and typically employ the inefficient autoregressive paradigm to generate semantic IDs. In this paper, we propose an end-to-end heterogeneous generative recommendation architecture called HGenPush. First, we design a hybrid user behavior understanding module that integrates multi-scenario and multi-perspective behaviors to capture precise user interest. Then, we design a dual-branch heterogeneous generative recommendation module that integrates video recommendation and author recommendation within a unified framework. In addition, to improve generation efficiency, we design a lightweight multi-token prediction method that discards the autoregressive paradigm. Finally, we design a user consumption preference alignment module, which leverages user feedback as reward signals to guide the model toward generating higher-quality content, thereby enhancing user experience and engagement. Through these designs, HGenPush simultaneously fulfills users' demands for high-quality content and trusted authors. We have deployed HGenPush on the push notification system of Kuaishou, a large-scale short-video platform, achieving a significant 0.181% increase in daily active users.

## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思想
- 采用 decoder-only 架构，输入混合用户行为序列（包括长期/短期feed行为、推送点击/发送行为、静态特征），通过因果自注意力理解用户兴趣，然后由双分支模块分别生成视频语义ID和作者混合语义ID，最后通过强化学习（RLVR）优化生成结果。

### 关键技术细节
1. **混合用户行为理解模块**  
   - 融合多场景行为：
     - 长期feed序列 (S_long)：基础兴趣，含视频语义嵌入、作者ID、话题标签、播放时长。
     - 短期feed序列 (S_short)：近期活跃行为。
     - 推送点击序列 (S_click)：推送特定行为模式，含通知类型特征。
     - 推送发送序列 (S_send)：上下文信息，增强生成合理性。
   - 静态特征（年龄、性别等）作为稳定锚点。
   - 在序列末尾添加特殊令牌 `[cls]_a` 和 `[cls]_v`，分别用于作者和视频分支的兴趣表示。
   - 堆叠16层 decoder-only 块（RMSNorm + SwiGLU FFN），最终得到全局兴趣表示 `h'_v` 和 `h'_a`。

2. **异构生成推荐模块**
   - **视频推荐分支**：采用**链式多令牌预测 (Chained-MTP)**  
     - 定义：使用用户兴趣表示 `h'[cls]_v` 作为稳定上下文，计算当前令牌前所有已生成令牌的累积嵌入 `∑_{j<k} Emb(j)(s_j)`，共同输入 FFN 预测第 k 个语义令牌。
     - 公式 (14)：`p(s_k) = softmax(FFN_k(h'[cls]_v + ∑_{j<k} Emb(j)(s_j)))`
     - 特点：单次前向传播并行预测所有令牌，无需自回归多步解码；通过累积嵌入保持语义依赖。
     - 损失：交叉熵，公式 (15)：`L_v-MTP = - (1/B) Σ_b Σ_k log(p(s_k_v_b))`

   - **作者推荐分支**：
     - 行为表征对齐：将作者特征编码后与用户兴趣表示 `h'[cls]_a` 通过 InfoNCE 对比学习对齐，捕捉偏好细微差异。
     - 作者语义ID构建：使用 RQ-Kmeans 将对齐后的作者嵌入离散化为语义ID，采用 SimVQ 稳定训练。
     - 混合语义ID：将作者语义ID（2个令牌）与目标视频的第一级语义ID拼接，形成长度 L_m = 3 的混合ID。
     - 同样使用 Chained-MTP 生成，损失 `L_a-MTP`。

3. **用户消费偏好对齐 (UCPA) 模块**
   - **奖励信号**：对推送点击后的会话内消费行为进行规则评分（有效播放+2，点赞/关注/分享+1，短播放-2，不感兴趣-5，举报-10）。
   - **优化算法：组序列重要性采样策略优化 (GSISPO)**  
     - 计算组内相对优势 `A_g` (公式21)。
     - 序列级重要性采样权重 `w_g` (公式23-24)：将各令牌似然比几何平均，再做裁剪 (clip 1-ε, 1+ε)。
     - 优化目标：`L_GSISPO = - (1/G) Σ_g sg(w_g) A_g log π_θ(s_v_g | h'[cls]_v)`，避免语义碎片化并保留梯度信号。
     - 不额外引入KL散度（由监督损失保证稳定）。

4. **联合训练与推理**  
   - 总损失：`L = L_v-MTP + L_a-MTP + L_codebook + L_GSISPO`  
   - 推理时使用 Beam Search 生成多个候选，直接送入排序阶段（跳过预排序）。
