## Abstract
Text-to-image personalization aims to generate a user-provided subject in novel scenes described by text. However, most existing methods encode subject identity (fidelity) and context (editability) through the same conditioning pathway, forcing the two to compete for attention-map resources. We refer to this phenomenon as conditioning entanglement and show that it induces a fidelity-editability trade-off. We further provide causal evidence by replacing the target subject token with a generic subject token, which produces shifts in attention allocation and corresponding changes in context adherence. To this end, we propose Decoupled Guidance (DeGu), a plug-and-play framework that routes subject identity and scene context through two independent guidance streams. We further introduce a spatial mixing mechanism that dynamically fuses these streams, ensuring each operates within its semantically relevant region without interference. Furthermore, DeGu can be readily applied to existing personalization methods without modifying the underlying backbone models, consistently improving the overall personalization performance while enabling inference-time control over the fidelity-editability balance, across diverse methods and backbones, including flow-matching Diffusion Transformers (DiTs).

## 1. 核心问题与整体含义（研究动机和背景）

- **研究动机**：文本到图像个性化生成中，现有方法（如 Textual Inversion、DreamBooth 等）普遍存在“保真度-可编辑性权衡”（fidelity-editability trade-off）。即：生成的图像越能保留主体身份（保真度），就越难以准确遵循文本提示中的场景、布局和属性（可编辑性）。这一瓶颈限制了个性化生成的实用价值。
- **核心问题识别**：论文诊断出该权衡的根源在于**条件纠缠（conditioning entanglement）**——所有现有方法都将主体身份（subject identity）和场景上下文（scene context）通过**同一个条件路径**（unified prompt）编码，导致两者在交叉注意力（cross-attention）中竞争有限的注意力资源。作者特别观察到一种称为**注意力崩塌（attention collapse）** 的现象：学习到的主体 token 吸收了绝大部分交叉注意力，而上下文 token（如颜色、物体、场景词）的注意力几乎被压制为零。
- **因果证据**：通过将目标主体 token 替换为通用类 token（如“dog”），注意力分布恢复均匀，上下文遵循度随之改善，证明注意力竞争是直接原因。
- **整体含义**：该工作揭示了现有方法内在的结构性缺陷，并提出通过解耦条件路径来彻底解决该权衡，使保真度和可编辑性成为两个独立可调的维度。

## 2. 方法论：核心思想、关键技术细节

### 2.1 核心思想
- 提出**解耦引导（Decoupled Guidance, DeGu）** 框架：将主体身份和场景上下文分别路由至**两个独立的引导流**，每个流拥有独立的交叉注意力键值对（key-value pairs），从而消除注意力竞争。
- 通过三个组件实现从训练到推理的全面解耦：Context-Agnostic Embeddings (CAE)、Decoupled Guidance Mixer (DGM)、Test-time Cross-Attention Masking (TCAM)。

### 2.2 关键技术细节

#### (a) Context-Agnostic Embeddings (CAE)
- **问题**：传统方法（如 Textual Inversion）在带有上下文的提示中训练主体 token，导致 token 已经编码了上下文共现信息，即使放入独立流也无法完全解耦。
- **解决方案**：将新的可学习 token `[S*]` 放在**只有填充 token 的序列**中训练，即序列为 `[SOT] [S*] [EOT] ... [EOT]`，完全不含上下文词汇。文本编码器输出条件 `c_f` 仅编码主体视觉特征，与任何语言上下文无关。
- **训练目标**：仅使用重建损失（MSE）优化 `[S*]` 的嵌入。该条件 `c_f` 被称为**保真度条件（fidelity condition）**。
- **上下文条件 `c_e`**：通过编码标准文本提示（如“a dog in the snow”）获得，不含任何可学习 token。

#### (b) Decoupled Guidance Mixer (DGM)
- **融合方式**：将标准 CFG 公式扩展为三部分：
```
˜ϵ(z_t) = ϵ(z_t) + γ_e (ϵ(z_t, t, c_e) - ϵ(z_t)) + γ_f (ϵ(z_t, t, c_f) - ϵ(z_t))
```
其中 `γ_e` 控制可编辑性强度，`γ_f` 控制保真度强度。
- **理论依据**：该公式等价于**专家乘积（PoE）** 分布下的分数组合，前提是 `c_f` 和 `c_e` 条件独立于 `z_t`。CAE 的设计天然鼓励这一独立性（`c_f` 无语言 token，`c_e` 无学习 token）。

#### (c) Test-time Cross-Attention Masking (TCAM)
- **问题**：仅靠 DGM 仍存在空间泄漏——`c_f` 的噪声预测中包含了参考图像的背景信息，可能污染上下文区域。
- **解决方案**：在推理时，从保真度流的交叉注意力图中动态提取**主体掩码（subject mask）**。具体方法：
  - 在保真度流的前向传播中，非主体 token（如 `[SOT]`）的注意力图自然集中于背景。通过反转（`1 - normalize(·)`）并二值化（高斯模糊 + Otsu 阈值）得到主体掩码 `M_t`。
  - 最终引导公式中加入空间分区：
```
˜ϵ(z_t) = ϵ(z_t) + γ_e[(ϵ(z_t, t, c_e) - ϵ(z_t)) ⊙ (1-M_t)] + γ_f[(ϵ(z_t, t, c_f) - ϵ(z_t)) ⊙ M_t]
```
- **优点**：完全在潜在空间内利用已有注意力图，无需外部分割模型，无需解码中间噪声，计算开销极小。

### 2.3 即插即用集成
- DeGu 可附加到任意现有个性化方法上：将现有方法的条件 `c_b`（如“a S* in the snow”）作为第三项加入，形成三流引导。CAE token 嵌入在预训练骨干上训练一次后，可在所有基线中复用，无需针对每种方法单独优化。

## 6. 主要结论与发现

1. **注意力崩塌是条件纠缠的直接证据**：在统一提示中，学习到的主体 token 平均吸收 86% 的交叉注意力，上下文 token 被抑制到 10% 以下；ACS 与保真度正相关、与可编辑性负相关。
2. **增加 token 容量反而加剧竞争**：将 TI 的 token 数从 1 增加到 5，保真度提升但可编辑性下降，注意力总占比从 89% 升至 96%，说明问题并非容量不足而是结构竞争。
3. **DeGu 打破了保真度-可编辑性权衡**：在 4 种骨干和 3 种范式上，DeGu 一致提高了综合得分（Overall），且保真度和可编辑性可通过 `γ_f` 和 `γ_e` 独立调节。
4. **即插即用有效**：将 DeGu 附加到现有方法（TI、P+、NeTI、DB、CoRe、DTI、MS-Diffusion）后，保真度显著提升，可编辑性保持或小幅改善，综合指标稳定增长。
5. **TCAM 优于外部分割**：相比 Grounded SAM 2，TCAM 在高噪声步下更稳定，且无需解码和外部模型，推理更快（4.5s vs 21.2s），MDINO 更高（0.8019 vs 0.6214）。
6. **多概念场景同样有效**：在 MS-Diffusion 基础上叠加 DeGu，同时保留两个主体的细节和场景符合度。

## 7. 优点

- **问题诊断深刻**：不仅发现 trade-off，还通过定量分析（ACS）和因果实验（替换 token）揭示了内在机制——注意力竞争，而非容量限制。
- **方法简洁有效**：三组件设计分别针对训练、推理、空间三个层面的纠缠，整体即插即用，无需修改骨干模型。
