## 摘要
个性化改变了模型对用户所说的话；我们表明它也能改变用于证明响应的推理轨迹。现代LLM通过存储用户属性、偏好和先前的上下文，然后将这些信息注入未来的提示来实现个性化交互。我们研究这种记忆是否会在没有单一真实答案的开放性问题中重塑推理。为了量化这种效应，我们引入了DRIFTLENS，一个无真实答案的框架，它将每个表达的推理步骤映射到一个价值类别，并测量问题的无记忆轨迹与注入用户属性记忆下的轨迹之间的差异。我们首先验证DRIFTLENS能够区分无内容的语用噪声与实质性的推理变化。在四个LLM和10个用户属性类别（包括年龄、职业和残疾）中，用户属性记忆在每个模型的语用噪声基准之上引发了中到大的推理漂移，即使最终答案保持流畅、切题且合理。然后，我们评估基于GRPO和DPO的后训练方法以减少漂移。两种方法都能减少漂移，但没有一种方法统一占优；对下游能力、有用性和指令遵循的影响取决于模型和奖励。这些结果表明，记忆引起的推理漂移是个性化语言模型的一个可测量的且仅部分缓解的失效模式。

## Abstract
Personalization changes what a model says to a user; we show that it can also change the reasoning trajectory used to justify the response. Modern LLMs personalize interactions by storing user attributes, preferences, and prior context, then injecting this information into future prompts. We study whether such memory reshapes reasoning on open-ended questions where no single ground-truth answer exists. To quantify this effect, we introduce DRIFTLENS, a ground-truth-free framework that maps each expressed reasoning step to a value category and measures divergence between a question's no-memory trajectory and its trajectory under injected user-attribute memory. We first validate that DRIFTLENS distinguishes content-free pragmatic noise from substantive reasoning changes. Across four LLMs and 10 user-attribute categories, including age, occupation, and disability, user-attribute memory induces medium-to-large reasoning drift above each model's pragmatic-noise floor, even when final answers remain fluent, on-topic, and plausible. We then evaluate GRPO- and DPO-based post-training methods for reducing drift. Both reduce drift, but neither uniformly dominates; effects on downstream capability, helpfulness, and instruction following are model-and reward-dependent. These results suggest that memory-induced reasoning drift is a measurable and only partly mitigated failure mode of personalized language models.

## 1. 核心问题与整体含义（研究动机和背景）

- **动机**：现代LLM（如ChatGPT）通过存储用户属性（年龄、职业、残疾等）并注入到对话中来个性化响应。这种“记忆”机制改变了回答的内容，但更关键的是，它可能悄无声息地**改变模型推理的轨迹**——即模型在生成回答前所依赖的价值权衡、优先级和论证路径。对于开放式、无真实答案的实用指导性问题（职业建议、道德困境等），这种推理漂移完全不可见，因为最终答案仍然流畅、切题且合理。
- **风险**：当用户属性与问题本身无关时（例如，询问工作纠纷时，模型却因为用户的年龄或残疾转向情感验证而非法律风险评估），这种漂移反映的是**不恰当的情景敏感度**而非合理的个性化适应。这种“符号漂移”在真实部署中难以被标准准确率指标捕捉，可能导致模型在关键决策支持场景中给出一致但偏倚的建议。
- **研究问题**：
  - RQ1：能否在无真实答案的情况下测量推理漂移？
  - RQ2：用户属性记忆是否跨LLM产生可测量的推理漂移？
  - RQ3：后训练方法能否减少这种漂移，代价是什么？

## 2. 方法论：核心思想、关键技术细节

- **核心思想**：提出**DRIFTLENS**框架，一种无真实答案的度量方法。它将每个回答的推理步骤映射到一个预定义的价值本体（value ontology）符号序列，然后比较同一问题下 **无记忆基线** 与 **注入用户记忆后** 的符号序列之间的差异，从而量化推理的“漂移”程度。
- **关键技术细节**：
  - **数据构造**：构建了422个严格筛选的问题（以及1,061个宽松版本），满足：①需推理权衡；②无唯一正确答案；③与提问者属性无关（persona-indifferent）；④独立可理解。来源包括Career Guidance、ETHICS、Medical QA等七个公共数据集，经LLM评分和人工审核过滤。
  - **价值本体**：经过78轮迭代优化，最终包含4个根类别（Social、Practical、Protective、Personal）和11个子类别。跨模型标注一致性从<50%提升至>83%，且对语义等价改写保持鲁棒（>83%）。
  - **扰动设置**：
    - 负控制：语用噪声（填充词、空白、标点），应不引起漂移。
    - 正控制：重大生活事件（童年逆境、丧亲、失业等），已知会重塑价值观。
    - 实验条件：10种用户属性记忆（性别、年龄、职业、残疾、外貌等）。
  - **度量指标**：
    - **DTW（动态时间规整）**：对齐两个符号序列的最小对齐成本，归一化到[0,1]。
    - **SRI（序列递归指数）**：结合编辑距离（顺序敏感）和Jensen–Shannon散度（分布差异），α=0.5加权。
  - **统计模型**：线性混合效应模型（固定效应：扰动类别；随机效应：问题截距），并使用**效应量** \(d_{rel} = (\mu_{treat}-\mu_{noise})/\sigma_{noise}\) 跨模型和属性进行比较，通过聚类bootstrap（B=10,000）计算95%置信区间和FDR校正的p值。

## 6. 主要结论与发现

- **DRIFTLENS有效**：负控制（语用噪声）不显著升高漂移；正控制（生活事件）显著升高；DTW与SRI高度相关（Spearman ρ>0.93），支持收敛效度。
- **用户属性记忆普遍引起漂移**：在所有4个模型上，10种属性均显著高于噪声基准，效应量中等到大（d_rel范围0.35–0.98）。跨性别和残疾是导致漂移最强的属性；教育、年龄等相对较弱。漂移存在于推理链而非最终答案，但会影响答案的有用性。
- **后训练可减少漂移但非一致**：
  - GRPO和DPO均能降低DTW，但无统一最优方法。DTW+SRI+fmt在Gemma2-2B上最好（0.186 vs 0.309基线），DPO(Tulu+SymPreference)在Qwen3-4B上最好（0.204）。
  - 漂移降低通常不带来能力显著退化（MMLU/GSM8K多数保留或改善），但效果依模型和奖励设计而异。
  - 格式奖励对GRPO漂移减少贡献大（消融发现仅漂移奖励效果有限）。
  - 人工评估表明DPO更可靠地降低“被干扰”而不损害有用性；GRPO在某些模型上可能过度优化有用性而牺牲鲁棒性。
- 结论：记忆诱导推理漂移是可测量且仅部分缓解的失效模式，需要在个性化部署中评估推理稳定性。

## 7. 优点

- **无真实答案场景的突破**：首次系统度量开放式推理任务中的记忆诱导漂移，不依赖ground truth，实用性强。
- **严格的方法论构建**：设计了负控制、正控制和正交性过滤，确保度量特异性；价值本体经过多轮优化和跨模型验证，可靠且可解释。
- **全面的统计框架**：使用混合效应模型、聚类bootstrap和效应量，量化了不确定性，避免假阳性。
- **后训练实验全面**：对比GRPO和DPO，包含多种奖励设计和数据组合，消融实验揭示关键因素（格式奖励、数据质量），并提供人工评估。
- **对风险属性的识别**：发现跨性别、残疾等属性引发的漂移最强，为审计和公平性提供了可操作信号。

## 8. 不足与局限

- **模型规模局限**：所有实验仅限于小/中等开源模型（≤4B参数），未在GPT-4、Claude等大型封闭模型上验证后训练效果，泛化性未知。
- **场景局限**：仅考虑单轮、固定、书面推理问题，未扩展到多轮对话、工具使用、长程交互或实际部署中的动态记忆系统。
- **本体固定**：价值本体可能遗漏跨文化或任务特定的细微差别，且标注依赖LLM（虽有>80%一致性，但仍存偏差风险）。
- **隐藏认知不可达**：DRIFTLENS测量的是外部表达的推理步骤，不是模型内部隐藏的认知过程，可能存在“表面漂移”。
- **资源消耗**：GRPO训练计算开销大，限制了实验范围；未提供详细的GPU训练时间，影响复现。
- **后训练未消除漂移**：即使最佳方法，DTW仍高于基线（如0.204 vs 0.242），表明漂移问题仅部分缓解。
